#CRUD API DE PRODUTO E CATEGORIA
<p>Esta é uma aplicação back-end, (API), feita na versão do .NET CORE 8 , CQRS e Mediator como Padrão Arquitectural </p>
<h6>Tecnologias Utilizadas:</h6>
<ul> 
   <li>.NET CORE 8</li>
   <li>CQRS como Padrão Arquitectural</li>
   <li>Mediator</li>
   <li>DDD</li>
   <li>Linguagem de Programação C#</li>
   <li>ORM Entity Frame Work Core 8 </li>
   <li> Fluent Validation </li>
   <li>Identity Entity Frame Work Core </li>
   <li>SGB SQL Server 2019, para criação do banco de dados</li>
</ul>
